# ゲーム のタイトル
*シン・こうかとん無双　改
## 実行環境の必要条件
* python >= 3.10
* pygame >= 2.1

## ゲームの概要
こうかとんを操作して爆弾をよけ敵機を撃ち落とすゲーム

## ゲームの実装
###共通基本機能
*第四回授業のコード
### 担当追加機能
* ゲームのレベルアップ
